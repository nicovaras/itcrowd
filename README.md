# itcrowd
It Crowd REST API Challenge

Available endpoints:

* users: /users
* person: /person
* movie: /movie
* aliases: /aliases

All endpoints have a detail view (like /users/[id]) and support GET, POST, HEAD, OPTIONS for list endpoints and GET, PUT, PATCH, DELETE, HEAD, OPTIONS for detail endpoints

Made with Django and Django REST Framework

Install dependencies with `pip install -r requirements.txt`
